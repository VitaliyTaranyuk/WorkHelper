## Проверено без замечаний
- Контроль доступа: чужой пользователь не читает задачу (404), не меняет статус (404 + понятное сообщение); эндпоинты без токена — 401.
- Валидация: title @NotBlank (включая пробелы), description ≤4096 — 400 с полевыми ошибками; malformed UUID — 404 без стектрейсов/SQL (error hygiene ок).
- DTO пользователей в задачах — без пароля/чувствительных полей; `creatorName: "admin"` у встреч — корректный displayName учётки test2 (не баг).
- XSS: `dangerouslySetInnerHTML`/innerHTML в кодовой базе отсутствуют; React экранирует.
- Комментарии/@упоминания, вложения, dev-панель PR, guard несохранённых изменений (ТП-34), чипы скрытых колонок, dnd-черновик доски — работают.
- Поллинг: доска 10s, списки/уведомления 30s, только активная вкладка — осознанная стратегия (ТП-47).
- Board: DnD-код и черновики колонок аккуратные; системные колонки защищены (ТП-32).
- Жизненный цикл задачи: создание → карточка → удаление (API) — согласован; кэш обновляется.
